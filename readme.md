# ストアサイト（WordPress 化前提・PHP テンプレ設計）

## 概要

CodeJump のデザイン課題をもとに、 **WordPress 化を前提とした構造設計・PHP テンプレート実装**を行ったストアサイトです。

単なる HTML/CSS の模写ではなく、将来的な CMS 運用を想定し、一覧ページ・詳細ページ・固定ページの切り分けや投稿ループを意識したマークアップを行っています。

---

## 制作の目的

- WordPress 移行を見据えた HTML 構造設計の練習
- PHP テンプレートでの実装に慣れること
- 「静的サイト → CMS 化」の実務フローを意識すること

---

## 工夫した点

- **HTML 作成時点でページの役割を整理**
  - 一覧ページ
  - 詳細ページ
  - 固定ページ（About / Company など）
- 投稿一覧になる想定箇所にはコメントで「ここからループ処理」と明示
- 共通パーツ（ヘッダー・フッター）を PHP で分割し、再利用しやすい構造に設計
- WordPress テンプレート化を想定したクラス設計・マークアップを意識

---

## 想定している WordPress 構成

- 投稿タイプ：商品（カスタム投稿想定）
- 一覧ページ：`archive-{post_type}.php`
- 詳細ページ：`single-{post_type}.php`
- 固定ページ：`page.php` または個別テンプレート
- 投稿表示：WordPress ループ（`have_posts()`, `the_post()`）を使用想定

※ 現在は PHP テンプレート段階ですが、 WordPress への組み込みを前提とした設計になっています。

---
